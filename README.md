# docker-pretty-images

### Description

Simple command line tool to output docker images in a pretty format


### Installation

Can be installed with uv via:

```bash
uv tool install git+https://github.com/j-p-m-7/docker-pretty-images
```


### Sample usage

```bash
$ docker-pretty-images
```
